# 🌟 Flex Yemen - منصة فلكس اليمنية المتكاملة

تطبيق **Flex Yemen** هو منصة تجارة إلكترونية ومحفظة مالية يمنية متكاملة، مصممة بأحدث تقنيات Flutter لتوفير تجربة مستخدم استثنائية (أكثر من 150 ملفاً برمجياً).

## 🚀 الميزات الرئيسية:
- **محفظة فلكس (Flex Wallet)**: 22 خدمة مالية وسداد فواتير (إنترنت، كهرباء، شحن ألعاب، تعليم).
- **السوق المفتوح (Flex Market)**: أكثر من 135 منتجاً مقسماً إلى (مزادات، عقارات، سيارات، إلكترونيات).
- **حديقة فلكس (Flex Garden)**: نظام ولاء وتفاعل لجمع النقاط واستبدالها بخصومات.
- **نظام الدردشة**: تواصل مباشر بين البائع والمشتري مع دعم الوسائط.
- **تصميم ذهبي (Golden UI)**: واجهة مستخدم فاخرة تدعم الوضعين الليلي والنهاري.

## 🛠️ متطلبات التشغيل:
1. تثبيت [Flutter SDK](https://docs.flutter.dev/get-started/install).
2. تثبيت [VS Code](https://code.visualstudio.com/) أو [Android Studio].
3. محاكي (Emulator) أو هاتف أندرويد حقيقي.

## 🏗️ خطوات البناء والتشغيل:
1. **تحميل المشروع**: قم بفك ضغط ملف `flex_yemen.zip`.
2. **تثبيت المكتبات**:
   ```bash
   cd flex_yemen
   flutter pub get
   ```
3. **تشغيل التطبيق**:
   ```bash
   flutter run
   ```
4. **استخراج ملف APK (النسخة النهائية)**:
   ```bash
   flutter build apk --release
   ```

## 📁 هيكل المشروع:
- `lib/models/`: نماذج البيانات للمنتجات والمحفظة.
- `lib/providers/`: إدارة الحالة (State Management) باستخدام Provider.
- `lib/screens/`: كافة الشاشات (أكثر من 80 شاشة فرعية).
- `lib/widgets/`: العناصر المشتركة والبطاقات الاحترافية.
- `lib/theme/`: نظام الألوان الذهبي والخطوط (Changa).
